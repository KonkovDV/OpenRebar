# Code Of Conduct

## Our Standard

This repository aims to be technically rigorous, respectful, and useful.

Contributors are expected to:

- critique code and decisions, not people
- provide reproducible evidence when reporting problems
- keep review discussion focused on correctness, safety, and maintainability
- avoid harassment, personal attacks, and hostile communication

## Unacceptable Behavior

- insulting or demeaning language
- deliberate intimidation or harassment
- publishing private information without permission
- derailing technical discussion into personal conflict

## Maintainer Responsibility

Maintainers may edit, remove, or reject comments, issues, or pull requests that
violate this document in order to keep collaboration safe and productive.

## Reporting

If conduct on a repository surface becomes unsafe, contact the maintainer privately
and include links or screenshots so the problem can be reviewed quickly.
