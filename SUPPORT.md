# Support

## Supported Topics

OpenRebar supports public engineering requests for:

1. reproducible reinforcement pipeline bugs;
2. schema/report contract regressions;
3. CI, packaging, and release-surface issues;
4. documentation and governance corrections;
5. ML module reproducibility and dependency-lock concerns.

## Not a Support Channel

1. public disclosure of vulnerabilities before private report;
2. emergency production support obligations;
3. confidential project model handling in public channels;
4. consulting requests without reproducible technical scope.

## Read First

1. README.md
2. CONTRIBUTING.md
3. SECURITY.md
4. docs/architecture.md

## Public Channels

1. GitHub Issues with templates for bugs/features.
2. Pull requests with explicit validation evidence.

## Response Model

OpenRebar follows best-effort open-source maintainer support.
