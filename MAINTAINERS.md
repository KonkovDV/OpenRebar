# Maintainers

## Current Maintainer Team

- Primary maintainer: KonkovDV
- Repository: https://github.com/KonkovDV/OpenRebar

## Responsibilities

Maintainers are expected to:

1. preserve clean architecture and port/adapter boundaries;
2. keep normative and geometric behavior traceable and testable;
3. review workflow and dependency changes for supply-chain risk;
4. keep report contracts stable and explicitly versioned;
5. keep README/governance docs aligned with shipped behavior.

## Escalation

Security incidents follow SECURITY.md private reporting path.
Release-critical decisions should receive maintainer approval before tagging.
